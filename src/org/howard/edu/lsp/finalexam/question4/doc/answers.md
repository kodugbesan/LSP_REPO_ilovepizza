My perspective on using AI in programming has changed because I see how helpful it is, especially when there is a time limit and a lot of questions to go through. Instead of getting stuck trying to understand what a question is asking, AI can help break it down quickly so I can actually focus on solving it. It also helps when something is confusing, because it can explain it in a simpler way.

In the future, I plan to use AI more as a tool to help me understand things rather than just give answers. For example, I would use it to break down long or confusing prompts, explain things step by step, and help me figure out what to do next when I’m stuck. I would also use it to check my approach or catch mistakes after I finish something.
